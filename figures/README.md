# Figures Directory

This directory contains publication-ready visualizations for the ICPE 2026 paper.

## Naming Convention
Use descriptive names with prefixes:
- `fig1_` - Main paper figures (numbered)
- `eda_` - Exploratory data analysis plots
- `model_` - Model performance visualizations
- `supp_` - Supplementary figures

## Requirements
- **Format:** PNG (300 DPI minimum for publication)
- **Size:** Typically 6-8 inches wide (double-column ACM format)
- **Style:** Consistent color palette, clear labels, readable fonts
