## Source

## Dataset Description
The Shaved Ice dataset contains approximately 3 years of hourly VM (virtual machine) demand data from Snowflake's cloud infrastructure, covering 2021-2024.

- **instance_type**: VM instance type/size
- **demand**: Number of VMs requested/needed
